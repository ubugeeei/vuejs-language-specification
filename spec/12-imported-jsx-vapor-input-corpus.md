# Imported JSX Vapor Input Corpus

## Scope

This annex lifts copied `vuejs/vue-jsx-vapor` JSX- and TSX-authored macro inputs into local static compiler test suites under `profile = "jsx-vapor"`.

These suites are snapshot-backed on purpose. They constrain the exact copied JSX Vapor macro and restructure output while this repository does not yet ship a default executable reference implementation for that profile.

## Notes

This annex is profile-scoped. It does not define base SFC syntax or template-parser conformance.

The current executable `jsx-vapor` surface covers macro lowering and parameter restructure. Directive spellings such as `$` dynamic arguments, `_` modifiers, and interop-mode switching remain provenance-tracked but are not yet executable local test suites.

Each suite in this annex binds:

- a copied upstream input artifact under `provenance/vendor/vuejs-vue-jsx-vapor/`
- a copied upstream Vitest snapshot under `provenance/vendor/vuejs-vue-jsx-vapor/expected-snapshots.pkl`
- an explicit `profile = "jsx-vapor"` marker so the evidence cannot be misread as base parser or syntax coverage
